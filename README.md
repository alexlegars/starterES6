# Webpack 3 Config


```bash
npm install
```
- ```npm start``` to run Dev Server with build files on cache | Running on 127.0.0.1:3000

- ```npm build``` to compile bundles

- ```npm watch``` to compile on change your bundles (No dev server)


This WebPack config contains :

* Webpack Dev Server
* ES6 with Babel
* SCSS with Sass
